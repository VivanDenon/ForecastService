# ForecastService
Webservice for forecasting of economic indicators

Targing platform: Linux

Using technology:
1) Python 3.10
2) Venv
3) TensorFlow 2.9
4) Django 
5) Datta Able Django 1.0.6
6) Matplotlib
7) Pandas
